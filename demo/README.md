# MobX Ajv Form Demo

> This demo project is using the build files from the `lib` directory instead of install it into npm dependecies.


### Install

1. In the main project directory:

  `npm install`

  `npm run build`


2. In the `demo` directory:

  `npm install`

  `npm run build`

  `npm run start`


### Development

1. In the main project directory:

  `npm install`

  `npm run build`


2. In the `demo` directory:

  `npm install`

  `npm run start:dev`
